aroz.org is a community of developers who value user experience and design in open source software. // aroz.org ist eine Gemeinschaft von Entwicklern, die Benutzererfahrung und Design in Open-Source-Software schätzen. // aroz.orgは、オープンソースソフトウェアにおいてユーザーエクスペリエンスとデザインを重視する開発者のコミュニティです。 // aroz.org是一個重視用戶體驗和設計的開源軟體社群




About aroz.org Projects // Über aroz.org Projekte // aroz.org プロジェクトについて // 關於 aroz.org 之開源專案

aroz.org, previously known as arozos.com, is the home of various open source projects spun off from ArozOS, the primary NAS OS originally designed for DIY NAS servers. With the help of university students, developers, and networking professionals all over the globe, we have been able to create several well-known projects. // aroz.org, früher bekannt als arozos.com, ist die Heimat verschiedener Open-Source-Projekte, die aus ArozOS hervorgegangen sind, dem ursprünglich für DIY NAS-Server konzipierten primären NAS-Betriebssystem. Mit Hilfe von Universitätsstudenten, Entwicklern und Netzwerkprofis auf der ganzen Welt konnten wir mehrere bekannte Projekte erstellen. // aroz.orgは、もともとDIY NASサーバー向けに設計された主要なNAS OSであるArozOSから派生したさまざまなオープンソースプロジェクトのホームです。世界中の大学生、開発者、ネットワーク専門家の助けを借りて、いくつかの有名なプロジェクトを作成することができました。 // aroz.org，以前稱為arozos.com，是各種 UX First 開源專案的啟發地，這些專案是從最初為 DIY NAS 伺服器設計的 NAS OS - ArozOS中分支出來的。在全球各地的大學生、開發人員和網絡專業人士的幫助下，我們已經創建了幾個知名專案。




Zoraxy

Zoraxy is an ultimate toolbox for your homelab, including a reverse proxy server, up-time monitoring, network tools, and more. // Zoraxy ist ein ultimatives Werkzeugkasten für Ihr Homelab, einschließlich eines Reverse-Proxy-Servers, Up-Time-Überwachung, Netzwerktools und mehr. // Zoraxyは、リバースプロキシサーバー、アップタイム監視、ネットワークツールなど、ホームラボ用の究極のツールボックスです。 // Zoraxy是您 Homelab 的便利工具箱，包括反向代理服務器、正常運行時間監控、網絡工具等功能。

ArozOS

ArozOS is an OS-like web-desktop interface that allows you to access your files and applications from anywhere. // ArozOS ist eine OS-ähnliche Web-Desktop-Oberfläche, mit der Sie von überall auf Ihre Dateien und Anwendungen zugreifen können. // ArozOSは、どこからでもファイルやアプリケーションにアクセスできるOSのようなWebデスクトップインターフェースです。 // ArozOS是一個類似作業系統的 Web 桌面界面，可讓您從任何地方訪問您的文件和應用程序。
